# PyLearn

PyLearn is a learner-facing GCSE Computer Science revision platform centered on
Python programming, exam-style practice, theory revision, and resilient AI
study support.

The project principles live in
[`.specify/memory/constitution.md`](./.specify/memory/constitution.md). Those
principles make curriculum accuracy, exam authenticity, learner-first scope,
and transparent AI fallback behavior non-negotiable.
