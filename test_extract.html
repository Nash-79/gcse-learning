Using a trace table to follow an algorithm
A programmer has written an algorithm to calculate the total cost of items in a shopping basket after applying a discount:

total = 0
count = 0
prices = [12, 8, 15, 5]

for i = 0 to 3
    total = total + prices[i]
    count = count + 1
next i

if total > 30 then
    discount = total * 0.1
    total = total - discount
endif

print(total)


Task (6 marks): Create a trace table to show how the values of the variables change as this algorithm executes. Show the value of each variable after each line executes. State the final output of the algorithm and explain what the algorithm does.

Levels of response (6 marks)

Level 1 (1-2): Attempts trace table but with significant errors; may not correctly identify which variables change at each step; final output incorrect; explanation missing or very basic.
Level 2 (3-4): Reasonable trace table tracking most variables correctly with minor errors; correctly identifies final output of 36 or close; some explanation including adding prices and applying discount but may lack detail.
Level 3 (5-6): Accurate, complete trace table with correct final output and clear explanation: Initial: total=0, count=0, prices=[12,8,15,5]. i=0: prices[0]=12, total=0+12=12, count=1. i=1: prices[1]=8, total=12+8=20, count=2. i=2: prices[2]=15, total=20+15=35, count=3. i=3: prices[3]=5, total=35+5=40, count=4. After loop: total=40, count=4. IF check: Is 40>30? TRUE. discount=40*0.1=4, total=40-4=36. Output: 36. What algorithm does: Calculates total cost of items in basket by summing array elements, then applies 10% discount if total exceeds £30. Items cost £12+£8+£15+£5=£40, exceeds £30 threshold, so £4 discount applied giving final total £36.

Indicative content: Tracking variables through iterations; array indexing with i; loop counter changes; conditional evaluation; percentage calculation; clear explanation of discount logic.

Step-by-step explanation:

Initial values: total=0, count=0, prices array contains [12, 8, 15, 5].

The FOR loop runs four times with i taking values 0, 1, 2, 3.

i=0: prices[0]=12. total=0+12=12. count=0+1=1.

i=1: prices[1]=8. total=12+8=20. count=1+1=2.

i=2: prices[2]=15. total=20+15=35. count=2+1=3.

i=3: prices[3]=5. total=35+5=40. count=3+1=4.

After loop: total=40, count=4.

IF statement: Is total > 30? Is 40 > 30? TRUE, so code inside IF executes. discount=40*0.1=4. total=40-4=36.

Final output: 36

What the algorithm does:

This algorithm calculates the total cost of items in a shopping basket and applies a percentage discount if the customer spends over a threshold. It uses a FOR loop to iterate through an array of item prices, adding each price to a running total and incrementing a counter to track how many items have been processed. After all items have been totalled, the algorithm uses selection (an IF statement) to check whether the subtotal exceeds £30. If the total is greater than 30, a 10% discount is calculated by multiplying the total by 0.1, and this discount amount is then subtracted from the total to give the final discounted price.

In this specific case, the basket contains items costing £12, £8, £15, and £5, giving a subtotal of £40. Since £40 is greater than the £30 threshold, the customer qualifies for the discount. The discount amount is £4 (10% of £40), which is subtracted to give a final total of £36 that is displayed to the customer.
